Pre-requisite:
1. Node.js and NPM should be installed on Windows
2. Visual Studio Code IDE should be installed from - "https://code.visualstudio.com/download" for Windows.

Steps to Run the test :-

1. Run "npm install cypress" command in Visual Sudio Code terminal.
2. Nw Run "npx cypress open" command (Cypress modal with spec.js files name should open)
3. Place the Trayio.spec.js file in Explorer>Cypress Folder>Integration>examples and save
4. Trayio.spec.js should apear in the modal openned earlier
5. Click the file to run the test
